Dedup & Routing Layers

1. Symbol-session dedup (primary noise reducer)

Only the FIRST BUY (or SHORT) alert per (user, symbol, direction, session_date) fires. Subsequent same-direction alerts that day are dropped regardless of alert_type. Collapses ETH-USD bouncing off EMA5/10/21/50/SMA50 into ONE alert instead of 5–11.

Exempt types (fire even if another BUY already fired today):

  • tv_open_reclaimed — Pine re-arms; multi-leg reclaim days
  • tv_open_wick_reclaim — distinct signal from open_held
  • tv_staged_pdh_break, tv_staged_pdl_reclaim — structural levels vs prior day
  • tv_staged_pwh_break, tv_staged_pwl_reclaim — structural levels vs prior week
  • tv_staged_pmh_break, tv_staged_pml_reclaim — structural levels vs prior month
  • tv_p{w,m}{h,l}_held / wick_reclaim — HTF support holds, daily-reset cadence

2. Identity dedup window

Same (symbol, direction, alert_type) within the window is suppressed. Per-type overrides:

  • tv_open_reclaimed: 90 min
  • tv_htf_proximity_*: 120 min
  • All others: 60 min default

3. Direction gates

  • SHORT: only fires on SPY / QQQ / AIQ / NDX (other-symbol SHORTs dropped at backend)
  • NOTICE: only fires on SPY / QQQ / AIQ / NDX by default. Exception: tv_htf_proximity_* fires on all stocks AND bypasses MUTE_NOTICE_ALERTS

4. Confluence twin suppression

When today's open ≈ PDH/PDL (within 0.3%), gap-up/gap-down scenarios fire both open_reclaimed AND staged_pdh_break on the same setup. The webhook suppresses the twin within a 5-min window. Telegram header upgrades to "Open + PDH confluence ↑" when both anchor to the same level.

4b. Cross-level confluence dedup NEW 2026-05-16

When a level alert (staged_*_break / staged_*_reclaim / staged_*_rejection / staged_*_failed_short) fires on a symbol, any same-side level alert that arrives within 30 min AND within 1% of the prior alert's entry price is suppressed. First-fires-wins. Prevents "PDL + PWL + PML all reclaim at the same recovery zone" flood.

Example: MSTR Friday — staged_pdl_reclaim @ $175.28 fires. If staged_pwl_reclaim arrives 5 min later @ $176 (PWL = $175.72), spread = 0.41% < 1% → suppressed.

Hold/wick_reclaim variants (pwh_held, pwl_wick_reclaim, etc.) are NOT included — they have their own per-session cadence. Configurable via LEVEL_CONFLUENCE_WINDOW_MIN and LEVEL_CONFLUENCE_PCT env vars.

5. Mute / Allowlist

  • MUTE_NOTICE_ALERTS=true (default): all NOTICEs blocked from Telegram except tv_htf_proximity_*
  • fire_vwap_alerts=false (default): VWAP NOTICEs disabled at Pine level
  • fire_proximity_alerts=false (default in ma-ema-daily): MA proximity NOTICEs disabled at Pine level
